# 単語例文ビューワー

## 概要
- 例文のみを表示し、ボタンでピンイン・日本語訳を表示
- 4段階評価（覚えた、微妙、全くダメ、重要でないから無視する）を記録
- 並べ替え（ドラッグ＆ドロップ）や進捗保存（ブラウザ内）

## 使い方
1. `npm install` で依存パッケージをインストール
2. `npm run dev` でローカルサーバーを起動
3. 画面上でJSONファイル（単語例文データ）を選択して閲覧・学習

## 必要なファイル
- `単語例文.json`（例：anki_chinese/期末テキスト/単語例文.json）

## 構成
- React + Vite
- ファイルは `scripts/word-viewer/` 以下に配置

---

## カスタマイズ
- 並べ替えや評価はブラウザのメモリ上のみ保存されます。
- 機能追加やデザイン変更は `src/WordList.jsx` を編集してください。
